# 2026 AI 智慧物流實戰養成班網站

這是一個可直接部署到 GitHub Pages 的靜態招生網站，包含：

- `index.html`：網站內容與區塊結構
- `style.css`：吉祥物綠白風格、LINE QR Code 聯絡卡片與 RWD 排版
- `script.js`：手機選單、倒數計時、滾動淡入效果

## 已填入資訊

- 240 小時培訓架構：第一階段 120 小時次產業 AI 應用課程、第二階段 120 小時企業專題實作
- 報名對象已獨立成一欄：18 至 35 歲待業青年、應屆畢業生、可配合完整培訓與企業專題者
- 兩班時程與師資
- Word 表一與表二完整課程內容
- 學習獎勵金、評量方式、實作工具環境
- 報名方式、應備文件、錄取通知與受訓保險說明
- 結訓與認證標準、請假退訓規定、參訓注意事項
- 承辦單位、合作企業與上課地點資訊

## 上線前建議調整

1. 將 `index.html` 中表單的 `action="#"` 改成正式 Google Form 或報名系統網址。
2. 上課地點已填入「國立雲林科技大學」。若正式簡章公布詳細教室，請更新 `#classes`、`#apply` 與 footer 區塊。
3. 若正式簡章公布電話、Email 或承辦窗口，請更新 `#apply` 與 footer 區塊。
4. 若開課日期異動，請同步更新倒數計時的 `data-target` 與班級時程文字。
